- [**Network Inspector improvements and Network Throttling for iOS Simulators**](/docs/features/network-inspector):
  On iOS Simulators, the Network Inspector now lets you throttle or disable the Internet connection of the simulated device.
  It can also now display the http(s) requests coming from Native Modules and Views, not just your application's JS code.

- **Expo SDK 55 support**
- Screenshots taken are now automatically copied to pasteboard.
- Allow AI Chats to access the device's Network Logs.
- Improved Storybook support, including custom configuration file path setting.
- Add "Action Button" support for iOS simulators.
- More stability improvements and bug fixes.
